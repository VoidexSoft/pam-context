<objective>
Wire document modification timestamps through the full ingestion pipeline to graph extraction, so that Graphiti's `add_episode()` uses the document's actual modification time as `reference_time` — producing correct bi-temporal `valid_at` timestamps on Neo4j edges.

Purpose: Closes EXTRACT-02 gap — currently all graph edges get `valid_at` set to ingestion time because `RawDocument` lacks a `modified_at` field and `getattr(raw_doc, "modified_at", None)` always returns `None`.

Output: `modified_at` flows from connector metadata through `RawDocument` -> `Document` ORM -> `add_episode(reference_time=...)` on both primary ingestion and sync-graph retry paths.
</objective>
