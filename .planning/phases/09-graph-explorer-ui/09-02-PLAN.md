**1. Create `web/src/hooks/useGraphExplorer.ts`** -- Central state management hook for the graph explorer:

State:
- `nodes: Node[]` (NVL Node objects on canvas)
- `rels: Relationship[]` (NVL Relationship objects on canvas)
- `selectedEntity: GraphNode | null` (clicked entity for sidebar details)
- `loading: boolean` (initial load state)
- `error: string | null`
- `entityCount: number` (total entity count for empty state detection)

Import types from `@neo4j-nvl/base` (`Node`, `Relationship`) and from `../api/client` (`GraphNode`, `GraphEdge`, `getGraphNeighborhood`, `getGraphEntities`, `getGraphStatus`, `getEntityHistory`, `EntityHistoryResponse`).

Core functions:
- `loadInitialGraph()` -- On mount, call `getGraphStatus()` to check entity count. If `total_entities > 0`, call `getGraphEntities({ limit: 1 })` to get the first entity, then `getGraphNeighborhood(entity.name)` to load its neighborhood as the starting view. If `total_entities === 0`, set empty state.
- `selectEntity(node: Node)` -- Set `selectedEntity` from the clicked node. Find matching `GraphNode` data from internal state. Call `getEntityHistory(node.caption)` to fetch temporal edge data for sidebar.
- `expandNeighborhood(node: Node)` -- Call `getGraphNeighborhood(node.caption)`. Merge returned nodes/edges into existing canvas state, deduplicating by UUID. Use `Set<string>` for existing node/edge IDs.
- `focusEntity(entityName: string)` -- Called from search. Load neighborhood for the named entity and replace canvas content.

Entity type color map (from RESEARCH.md):
```typescript
const ENTITY_TYPE_COLORS: Record<string, string> = {
  Person: "#6366f1",
  Team: "#8b5cf6",
  Project: "#3b82f6",
  Technology: "#10b981",
  Process: "#f59e0b",
  Concept: "#ec4899",
  Asset: "#6b7280",
};
```

Conversion functions (use `useMemo` for stable references):
- `toNvlNode(graphNode, connectionCount)` -- Maps GraphNode to NVL Node with color by entity_type and size by connection count. Size formula: `Math.max(20, Math.min(50, 20 + connectionCount * 5))`.
- `toNvlRel(graphEdge, nameToUuid)` -- Maps GraphEdge to NVL Relationship. Build a `Map<string, string>` (name -> UUID) from all nodes for `from`/`to` resolution. Set caption to `relationship_type`.

Return: `{ nodes, rels, selectedEntity, selectedEntityHistory, loading, error, entityCount, selectEntity, expandNeighborhood, focusEntity, deselectEntity }`
